# Password Strength Checker

A cybersecurity tool built with Python to evaluate password security.

## Features

- Detects weak passwords
- Checks password length
- Detects uppercase and lowercase letters
- Detects numbers and special characters
- Flags common passwords used in breaches

## Technologies

- Python
- Regex (Regular Expressions)

## Usage

Run the program:

python password_checker.py

## Example

Enter password: H@ck3rSecure2026

Output:

Password Strength: Strong Password
